This folder contains python programs and data file, for training, testing, and predicting the submisison file. 
Instructions for reproduction of submisison file:
1. Download this folder
2. Set up the environment exactly the same as stated in the file "system_requirement.txt".
3. Run file "ssa_perform.py", to generate "linear.csv", which is crucial for Theil-sen regression in the main.py file.
4. Run "main.py"
5. After this step, there will be some images created in the same folder, which are the prediction-truth comparison in different cross-validation
6. In addition, there is a "report.txt" created, showing the result of testing and explaining the model.
